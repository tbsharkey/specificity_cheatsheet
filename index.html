Specificity Cheatsheet

What is CSS Specificity?

When multiple selectors affect an element specificity provides a way for browsers to render the most appropiate property values. By weighting various types of selectors precedence to the value with the highest ranking. Understanding specificity will allow a developer to work effectively with multiple selectors on the same element.

Specificity is ranked in descending order by:

Selector: Example
ID #something {color: red}
Class .anything {color: blue}
Type

How to Override Specificity

Using !important within the CSS declaration will force that declaration to be used and ignore any others regardless of selector specificity. It is generally considered a bad practice to utilize.
